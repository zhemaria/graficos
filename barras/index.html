Construye un gráfico de barras paso a paso

Organiza los datos, levanta las barras y aprende a leerlas.

¿Qué vas a aprender?

A organizar datos en una tabla, a construir un gráfico de barras donde la altura de cada barra representa su cantidad, y a leer el gráfico para responder preguntas sencillas. Avanza un paso a la vez: cada paso se abre cuando completas el anterior.

Cada tema reúne cuatro categorías. Elige uno para comenzar.

Escribe cuántos hay en cada categoría. Puedes usar los números de ejemplo o cambiarlos por los tuyos.
