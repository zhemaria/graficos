La media ponderada como punto de equilibrio

Cada puntaje se coloca sobre la regla. Su frecuencia son las pesas apiladas encima. La media ponderada es el punto donde toda la balanza se equilibra.

Cada pesa = un puntaje obtenido Media ponderada (punto de equilibrio) Promedio simple (sin frecuencias)

Mostrar el promedio simple (tratar todos los puntajes por igual)

Ajuste las frecuencias

Use los botones para indicar cuántas veces se obtuvo cada puntaje. Observe cómo el punto de equilibrio se desplaza hacia donde se acumulan las pesas.

El cálculo, paso a paso

Cada puntaje se multiplica por su frecuencia; luego se suma todo y se divide entre el total de pesas.
Puntaje (x) Frecuencia (w) w · x
Totales 0 0
Media ponderada = (suma de w · x) ÷ (suma de w) = 0 ÷ 0 = 0
Idea clave. El promedio simple siempre cae en el centro de los puntajes (30), porque ignora cuántas veces apareció cada uno. La media ponderada, en cambio, se inclina hacia el lado donde hay más frecuencias: si la mayoría de los estudiantes sacó puntajes bajos, el equilibrio se desplaza a la izquierda; si sacaron puntajes altos, se desplaza a la derecha.
